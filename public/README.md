# Public Assets Folder

## College Building Image
Please save your college building image as `college-building.jpg` in this folder.

The image will be used as the hero background on the homepage.

Recommended image specifications:
- Format: JPG, PNG, or WebP
- Resolution: 1920x1080 or higher
- File size: Optimized for web (< 500KB recommended)
- Aspect ratio: 16:9

## How to Add the Image
1. Save your college building photo as `college-building.jpg`
2. Place it in this `public` folder
3. The website will automatically use it as the hero background

Alternatively, you can use any other name and update the path in:
`src/pages/LandingPage.jsx` (line with `backgroundImage: url(...)`)
